# Godot Project

A simple Godot 4.2 project with a basic scene setup.

## Features

- Basic Node2D scene with a label
- Simple GDScript example
- Standard Godot project structure

## Getting Started

1. Open the project in Godot 4.2 or later
2. Run the Main.tscn scene
3. You should see "Hello, Godot!" displayed on the screen

## Project Structure

- `project.godot` - Main project configuration file
- `Main.tscn` - Main scene file
- `Main.gd` - Main script file
- `icon.svg` - Project icon
